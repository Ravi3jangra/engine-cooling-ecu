# Engine Cooling System (ECU)

## Overview
Developed a thermal management system using MATLAB/Simulink and Stateflow for engine temperature control.

## Features
- Finite State Machine (FSM) control
- 3-state fan operation (OFF, Medium, FULL)
- Temperature-based control logic

## Technologies Used
- MATLAB
- Simulink
- Stateflow

## Working
- Temperature input triggers FSM states
- Fan speed adjusts dynamically
- System maintains optimal temperature range

## Applications
- Automotive ECU systems
- Thermal management systems
